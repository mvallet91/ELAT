ELAT: edX Logfile Analysis Tool

Simple browser-based tool to process a course's events from logfiles into a local database for analysis

Learn more

What is ELAT?

ELAT is a simple, browser-based, fully local processing tool that can take the logfiles with events produced by edX and generate a database that can be downloaded as csv files, along with some basic indicators of students in a given edX course. You can start using ELAT simply by going to the homepage and uploading metadata files and logfiles (uploading to your own browser, there is no server involved). The code is fully available here.

Easy and Private

Get started by uploading the metadata files of the course, then the logfiles in the corresponding time period. Once processed, the csv files for each type of session can be downloaded. Everything stays in your computer and everything can be deleted with a single button.

Session Focused

ELAT considers a student's session as a series of events of a certain type, for example a video interaction session starts when a student opens a video, and the session contains the number of pauses, rewinds, fast forwards, etc., until the video is closed

Primary Indicators

From the information extracted from the logfiles, ELAT automatically generates a few general indicators, such as the number of students per day, and the duration of their sessions.
